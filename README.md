Introduction to Web Programming Project
Overview

This project was created as part of the Introduction to Web Programming course. The goal of the project is to demonstrate basic web development skills using HTML, CSS, and JavaScript.

Technologies Used

HTML5 – for structuring the web pages

CSS3 – for styling and layout

JavaScript – for client-side interactivity

PostgreSQL – for storing and managing the application data

Purpose of the Project

The purpose of this project is to practice:

-Building a basic web interface

-Writing SQL queries
